3.1 Estilos en CSS

CSS (Cascading Style Sheets) permite dar estilos a tu HTML: colores, tamaños, fondos, fuentes y más.

Aplicando estilos

Hay varias formas de aplicar estilos:

1. Estilo en línea (inline)

<p style="color:red; font-size:20px;">Hola</p>

Hola, estilo en línea

2. Estilo interno (internal)

Se coloca dentro de la etiqueta <style> en el <head>.


<style>
p { color: blue; font-size:18px; }
</style>

Hola, estilo interno

3. Estilo externo (external)

Se crea un archivo estilos.css y se enlaza con <link>.

<link rel="stylesheet" href="estilos.css">

Es la forma más recomendable para proyectos grandes.

Hojas de estilo en cascada (Cascading)

Si un elemento tiene varios estilos aplicados, se aplican en este orden:

  1. Estilo por defecto del navegador
  2. Estilo externo
  3. Estilo interno
  4. Estilo en línea

Esto se llama la cascada en CSS.

Seudoclases

Permiten aplicar estilos a estados especiales.

  • :hover → cuando el mouse pasa sobre el elemento
  • :active → cuando se hace clic
  • :first-child → primer hijo de un contenedor
  • :last-child → último hijo
Ejemplo hover

Buenas prácticas

  • Usar clases y selectores de forma clara
  • Evitar estilos en línea cuando sea posible
  • Organizar CSS en archivos externos
  • Usar nombres descriptivos para clases e IDs
  • Probar los estilos en diferentes navegadores
