CSS (Cascading Style Sheets) se utiliza para dar estilo y diseño a las páginas web.

3.1 Estilos

Aplicando estilos

Hay 3 formas de aplicar CSS:

  • En línea: usando style=""
  • Interno: dentro de <style>
  • Externo: archivo .css
