3.2 Otros selectores útiles en CSS

En CSS existen selectores más avanzados que permiten apuntar a elementos de manera muy precisa, basándose en atributos, posición o relación con otros elementos. Usarlos correctamente facilita el mantenimiento del código y evita agregar clases innecesarias.

1. Selectores de atributo

Estos selectores permiten aplicar estilos a elementos HTML en función de sus atributos, por ejemplo type, href o src. Son muy útiles cuando no quieres añadir una clase para cada elemento.

[atributo]

Selecciona todos los elementos que tengan un atributo específico, sin importar su valor. Por ejemplo:

input[type] { border: 2px solid blue; }

Este estilo aplica a cualquier <input> que tenga un atributo type, como text, email o password. Esto evita tener que usar clases en cada input.

[atributo="valor"]

Selecciona elementos cuyo atributo tenga exactamente un valor específico. Por ejemplo, si solo queremos afectar un enlace a Google:

a[href="https://google.com"] { color: red; font-weight: bold; }

Esto evita afectar otros enlaces que puedan apuntar a Facebook, Gmail o cualquier otra página.

[atributo^="valor"]

Selecciona elementos cuyo atributo comience con un valor determinado. Por ejemplo:

a[href^="https://g"] { background-color: yellow; }

Esto seleccionará todos los enlaces cuyo href empiece con la letra g, como Google o Gmail.

[atributo$="valor"]

Selecciona elementos cuyo atributo termine con un valor específico, útil para extensiones de archivos:

a[href$=".jpg"] { color: green; font-weight: bold; }

Esto aplica solo a enlaces que terminan en ".jpg", ignorando PDF, PNG u otros archivos.

[atributo*="valor"]

Selecciona elementos cuyo atributo contenga un valor en cualquier parte. Muy útil para buscar palabras dentro de atributos largos:

a[href*="insta"] { text-decoration: underline; color: purple; }

Esto afectará cualquier enlace que contenga la palabra "insta", como Instagram.

2. Combinaciones de selectores avanzados

Además de los selectores de atributo, CSS permite combinaciones que dependen de la relación entre elementos.

Selector descendiente (espacio)

Aplica estilos a todos los elementos dentro de otro elemento. Por ejemplo, si quieres cambiar todos los párrafos dentro de un div:

div p { color: brown; }

Selector hijo directo (>)

Selecciona solo los hijos directos de un elemento, ignorando los descendientes más profundos:

div > p { font-weight: bold; }

Buenas prácticas

  • Usar selectores de atributo y combinaciones solo cuando sea necesario.
  • Evitar selectores demasiado complejos, pueden dificultar mantenimiento.
  • Combinar con clases para mayor claridad y control.
  • Probar en diferentes navegadores, algunos selectores avanzados se comportan distinto.
  • Documentar tus selectores complejos con comentarios en CSS.
