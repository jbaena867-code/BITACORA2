2.2 Contenido en HTML

En HTML, el contenido es todo lo que el usuario ve en la página: texto, enlaces, imágenes, listas, tablas, etc.

Texto

HTML permite estructurar el texto usando diferentes etiquetas.

Encabezados

Van desde h1 hasta h6:

h1 - Título principal

h2 - Subtítulo

h3 - Sección

Párrafos

Se crean con <p>.

Este es un párrafo de ejemplo.

Formato de texto

  • <strong>: negrita
  • <em>: cursiva
  • <mark>: resaltado
  • <small>: texto pequeño

Saltos y líneas

Se usa <br> para saltos de línea.

Línea 1
Línea 2

Enlaces

Los enlaces permiten navegar entre páginas.

Enlace básico

<a href="https://www.google.com">Ir a Google</a>
Abrir Google

Atributos importantes

  • href: destino
  • target="_blank": abre en nueva pestaña
  • title: descripción

Enlaces internos

Sirven para moverse dentro de la misma página.

Ir a sección Listas

Imágenes

Se insertan con <img>.

<img src="imagen.jpg" alt="Descripción">

Atributos importantes

  • src: ruta de la imagen
  • alt: descripción (accesibilidad)
  • width / height: tamaño

Tablas

Se usan para mostrar datos organizados.


<table>
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
  </tr>
</table>
Nombre Edad
Ana 22
Carlos 30

Etiquetas importantes

  • <table>: tabla
  • <tr>: fila
  • <td>: celda
  • <th>: encabezado

Atributos globales

Se pueden usar en cualquier elemento:

  • id
  • class
  • style
  • title

Buenas prácticas

  • Usar etiquetas correctas para cada contenido
  • Agregar siempre alt en imágenes
  • Mantener código limpio y ordenado
