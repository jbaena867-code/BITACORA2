2.1 Estructura en HTML

La estructura HTML define cómo se organiza el contenido de una página web.

Tipo de documento

Todo documento HTML debe comenzar con:

<!DOCTYPE html>

Esto indica al navegador que estamos usando HTML5.

Estructura básica

Una página HTML tiene esta estructura mínima:


<!DOCTYPE html>
<html>
    <head>
        <title>Título</title>
    </head>
    <body>
        Contenido
    </body>
</html>
  • html: contiene toda la página
  • head: información (no visible)
  • body: contenido visible

Elementos estructurales

HTML5 introduce etiquetas para organizar mejor el contenido:

  • <header>: encabezado
  • <nav>: menú de navegación
  • <main>: contenido principal
  • <section>: sección
  • <article>: contenido independiente
  • <aside>: contenido secundario
  • <footer>: pie de página

Ejemplo visual de estructura

Header (Encabezado)
Nav (Menú)
Section (Sección)
Article (Artículo)
Aside (Contenido secundario)
Footer (Pie de página)

Atributos globales

Son atributos que se pueden usar en casi todas las etiquetas:

  • id: identifica un elemento único
  • class: agrupa elementos
  • style: aplica estilos directos
  • title: muestra información al pasar el mouse

Buenas prácticas

  • Usar etiquetas semánticas (header, main, footer)
  • Mantener una estructura clara
  • No abusar de divs innecesarios
  • Separar HTML y CSS
