3.2 Referencias en CSS (Selectores)

Los selectores permiten apuntar a elementos específicos del HTML para aplicar estilos.

1. Selectores básicos

Selector de etiqueta

p { color: red; }

Este texto es un párrafo seleccionado por su etiqueta

Selector de clase

.miClase { background: yellow; }

Este texto está dentro de una clase "miClase"

Selector de ID

#miId { font-weight: bold; }

Este texto tiene ID "miId"

2. Selectores combinados

Etiqueta + clase

p.texto { color: purple; }

Texto de párrafo con clase "texto"

Etiqueta + ID

div#contenedor { background: lightgray; }

Este div tiene ID "contenedor"

Descendientes

div p { font-style: italic; }

Este párrafo está dentro de un div

3. Seudoclases

Permiten aplicar estilos a estados especiales de los elementos.

  • :hover → al pasar el mouse
  • :active → al hacer clic
  • :first-child → primer hijo de su contenedor
  • :last-child → último hijo de su contenedor
  • :nth-child(n) → enésimo hijo

4. Otros selectores útiles

  • [atributo]: selecciona por atributo
  • [atributo="valor"]: igual a valor
  • [atributo^="valor"]: empieza con valor
  • [atributo$="valor"]: termina con valor
  • [atributo*="valor"]: contiene valor

Buenas prácticas con selectores

  • Usar clases y IDs descriptivos
  • Evitar usar demasiados selectores complejos
  • Priorizar selectores de clase para consistencia
  • Combinar selectores cuando sea necesario para especificidad
  • Documentar estilos en CSS externo
