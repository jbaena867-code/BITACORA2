3.2 Seudoclases en CSS

Las seudoclases permiten aplicar estilos a elementos según su estado o posición, sin necesidad de añadir clases adicionales.

1. Seudoclases de estado

Estas seudoclases aplican estilos dependiendo de la interacción del usuario:

:hover

Se activa cuando el usuario pasa el mouse sobre un elemento.

Pasa el mouse aquí Otro enlace

:active

Se activa cuando el elemento está siendo clickeado.

Haz clic aquí

:focus

Se aplica cuando un elemento tiene el foco (ej: un input seleccionado).

2. Seudoclases de posición

Permiten seleccionar elementos según su posición dentro de un contenedor.

:first-child

Selecciona el primer hijo de su contenedor:

  • Primer elemento
  • Segundo elemento
  • Tercer elemento

:last-child

Selecciona el último hijo de su contenedor:

  • Elemento 1
  • Elemento 2
  • Último elemento

:nth-child(n)

Selecciona elementos según su posición (n puede ser número o fórmula).

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemento 4
  • Elemento 5

Buenas prácticas con seudoclases

  • Usar seudoclases para evitar agregar clases innecesarias.
  • Combinar seudoclases con selectores de clase para mayor precisión.
  • Evitar abusar de :nth-child muy complejos, dificulta mantenimiento.
  • Probar en distintos navegadores, algunas seudoclases avanzadas pueden comportarse diferente.
