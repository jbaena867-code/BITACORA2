2.3 Formularios en HTML

Los formularios permiten recoger información del usuario y enviarla a un servidor.

Definición

Un formulario se crea con la etiqueta <form>.


<form action="servidor.php" method="POST">
    <input type="text">
</form>
  • action: a dónde se envían los datos
  • method: cómo se envían (GET o POST)

Envío del formulario

Cuando el usuario hace clic en "Enviar":

  • Los datos se envían al servidor
  • Se usa action para la URL
  • Se usa method="GET" o POST

Diferencias

  • GET: datos visibles en la URL
  • POST: datos ocultos (más seguro)

Validaciones

HTML permite validar datos automáticamente:

  • required: obligatorio
  • min / max: límites
  • maxlength: máximo de caracteres
  • pattern: expresión regular

Atributos importantes

  • name: nombre del campo (clave para enviar datos)
  • placeholder: texto de ayuda
  • value: valor inicial
  • disabled: desactiva el campo

Buenas prácticas

  • Usar etiquetas <label>
  • Validar datos siempre
  • Usar name en inputs
  • Evitar formularios muy largos
  • Diseñar formularios claros
