Capítulo 2 — HTML

HTML (HyperText Markup Language) es el lenguaje que se utiliza para crear la estructura de las páginas web.

2.1 Estructura

Tipo de documento

El documento HTML comienza con <!DOCTYPE html>, que indica al navegador que se está usando HTML5.

Elementos estructurales

Los elementos estructurales organizan la página:

  • <header>: encabezado
  • <nav>: navegación
  • <main>: contenido principal
  • <section>: secciones
  • <footer>: pie de página

Ejemplo:

Encabezado
Sección de contenido
Pie de página

Atributos globales

Se pueden usar en casi todas las etiquetas:

  • id: identificador único
  • class: agrupar elementos
  • style: aplicar estilos
  • title: texto al pasar el mouse
