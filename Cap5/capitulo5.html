Capítulo 5 — Diseño Web Adaptable

El diseño adaptable (Responsive Design) permite que una página se vea bien en cualquier dispositivo: celular, tablet o computadora.

5.1 Web móvil

Meta viewport

Esta etiqueta es esencial para que el diseño funcione en móviles:

<meta name="viewport" content="width=device-width, initial-scale=1.0">
Este contenido se adapta al tamaño de la pantalla

Media Queries

Permiten aplicar estilos según el tamaño de pantalla.

  • Desktop: más de 900px
  • Tablet: menos de 900px
  • Móvil: menos de 600px
